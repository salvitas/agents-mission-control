# ACRA Submission Checklist

## Pre-Submission
- [ ] Name availability checked for both entities
- [ ] Backup names prepared
- [ ] Shareholder/director details verified exactly as per ID docs
- [ ] Registered office address confirmed
- [ ] Principal activity wording confirmed with SSIC alignment
- [ ] FYE confirmed (31 Dec)
- [ ] Paid-up capital confirmed

## Name Reservation
- [ ] Arros Pte. Ltd. submitted
- [ ] SVK Pte. Ltd. submitted
- [ ] Approval/queries resolved

## Incorporation Filing
- [ ] Constitution selected and attached
- [ ] Director consent signed
- [ ] Shareholder particulars complete
- [ ] Secretary appointment workflow initiated
- [ ] Filing submitted for Arros
- [ ] Filing submitted for SVK

## Post-Approval Artifacts
- [ ] UEN received (Arros)
- [ ] UEN received (SVK)
- [ ] Biz Profile downloaded (both)
- [ ] Certificate/incorporation proof archived (both)
- [ ] Constitution archived (both)

## Critical Guardrail Verification
- [ ] No SVK shareholding in Arros (first 3 years)
- [ ] No accidental cross-entity ownership links in filed records
